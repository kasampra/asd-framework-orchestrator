# QA Workflow

## Prerequisites
- Read `AGENTS.md` in project root.
- All build phases (3–5) must be complete.

## Steps

1. **Run backend tests**:
   ```bash
   cd backend && pytest tests/ -v --cov=app --cov-report=term
   ```
   Target: >80% coverage.

2. **Run frontend build**:
   ```bash
   cd frontend && npm run build
   ```
   Must pass with zero TypeScript errors.

3. **End-to-end browser test**:
   - Navigate to `http://localhost:3000`
   - Submit form with sample project brief
   - Confirm AI-generated requirements render correctly
   - Click PDF download → confirm file downloads

4. **Write missing tests** to reach >80% coverage if needed.

5. **Produce test report** (`.agent/artifacts/test-report.md`):
   - Pass/fail summary
   - Coverage percentage
   - Any failing tests with details

## Output
`.agent/artifacts/test-report.md`
